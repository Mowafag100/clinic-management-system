# نظام إدارة العيادة الطبية

![ clinic-management-system ]

هذا المشروع هو **مشروع تخرج** مقدم من الطالب **موفق  فوزي يوسف محمد**  
بكلية **الأفق للعلوم والتكنولوجيا** – قسم تقنية المعلومات  
لنيل درجة **الدبلوم في تقنية المعلومات**.

## 📋 وصف المشروع
نظام ويب متكامل لإدارة العيادات الطبية، يسمح بتسجيل المرضى، حجز المواعيد، وإنشاء سجلات طبية. تم تطويره باستخدام **HTML, CSS, JavaScript** خالص (بدون مكتبات خارجية) مع الاعتماد على **Local Storage** لتخزين البيانات.

## ✨ المميزات
- نظام تسجيل دخول متعدد المستخدمين (مسؤول / مستخدم عادي).
- لوحة تحكم تعرض إحصائيات سريعة (عدد المرضى، مواعيد اليوم، السجلات).
- إدارة المرضى (إضافة، تعديل، حذف، بحث).
- إدارة المواعيد مع تصنيف الحالة (مجدول، منتهي، ملغي).
- إدارة السجلات الطبية لكل مريض.
- إدارة المستخدمين (خاص بالمسؤول).
- تقارير متنوعة: تقرير المرضى، تقرير المواعيد، تقرير طبي.
- واجهة مستخدم عربية بالكامل وتصميم متجاوب مع جميع الأجهزة.

## 🔧 التقنيات المستخدمة
- HTML5
- CSS3 (مع متغيرات CSS وتصميم متجاوب)
- JavaScript (ES6)
- Local Storage لتخزين البيانات محلياً

## 📥 طريقة التشغيل
1. حمّل الملفات من هذا المستودع.
2. افتح ملف `index.html` في أي متصفح حديث (كروم، فايرفوكس، إيدج).
3. ابدأ باستخدام النظام – سيتم تهيئة التخزين المحلي تلقائياً.

## 🚀 نشر المشروع على الإنترنت (GitHub Pages)
لجعل المشروع متاحاً كموقع مباشر:
1. اذهب إلى إعدادات المستودع (Settings).
2. اختر قسم **Pages**.
3. من **Branch** اختر `main` والمجلد `/root`، ثم Save.
4. بعد دقائق، سيكون موقعك متاحاً على الرابط:  
   `https://Mowafag100.github.io/clinic-management-system`
